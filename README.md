LLEGAREMOS AL EXAMEN DE BASH SCRIPTING Y HAREMOS LO SIGUIENTE

1. PASO:

```
git clone https://github.com/GorosX/examenBash.git
```

2. PASO:

```
cd examenBash
```

3. PASO: En vez de "scriptbash.sh" pondremos el script que queremos ejecutar

```
chmod +x scriptbash.sh
```

4. PASO: En vez de "scriptbash.sh" pondremos el script que queremos ejecutar

```
./scriptbash.sh
```
